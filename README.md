## 🚀 About Byapar Invoice

**Byapar Invoice** transforms your phone into a billing machine.

Key features:
- Create GST & Non-GST invoices in seconds
- Smart stock & inventory management
- Offline billing with auto-sync
- Secure cloud backup
- Customer payment tracking
- Professional branded invoices

Built by the **ByaparMate** family with a focus on simplicity and speed.

## 🐞 How to Report a Bug

Before reporting:
1. Make sure the issue is not already reported
2. Update the app to the latest version

When creating a bug report, please include:
- App version
- Device & Android version
- Steps to reproduce the issue
- Expected behavior
- Screenshots or screen recording (if possible)

👉 Use the **Bug Report** issue template.

---

## 💡 Request a Feature

Have an idea that can make billing easier or faster?

When requesting a feature:
- Explain the problem you are facing
- Describe how the feature would help your business
- Mention your business type (Kirana, Medical, Construction, etc.)

👉 Use the **Feature Request** issue template.

---

## 📌 Issue Labels We Use

- `bug` – Something not working correctly
- `feature-request` – New feature suggestion
- `enhancement` – Improvement to existing feature
- `ui/ux` – Design or usability feedback
- `high-priority` – Critical issues affecting billing
- `needs-info` – More details required
